# 国内外研究现状编写教程（精简模板）

本教程用于 `1.2 国内外研究现状` 的写作，推荐流程是：

1. 先整理一个 Markdown 文献笔记文件（含 BibTeX 键 + 你的读后摘要）
2. 再将该笔记交给 AI 生成正文初稿
3. 最后人工校对逻辑、术语、引用与行文风格

---

## 一、推荐工作流

### 步骤 1：先做“文献笔记 Markdown”

每篇文献至少记录以下信息：

- `citekey`（与 `references.bib` 一致）
- 研究问题（一句话）
- 方法核心（1-2 句话）
- 主要结论（1 句话）
- 你要用它支撑哪一句话（用途标记）

建议格式如下：

```md
## [citekey]
- 问题：
- 方法：
- 结论：
- 在正文中的用途：
```

### 步骤 2：按段落目标组织素材

推荐将“雷达生命体征感知研究现状”组织为 5 段：

1. 背景概述（领域意义 + 雷达优势）
2. 基本原理（极简，不做推导）
3. 传统研究主线（呼吸/心率/分离/多人）
4. ECG 重建研究（本节重点，建议 40%~50% 篇幅）
5. 研究不足与本文切入点（过渡到你的方法）

### 步骤 3：让 AI 生成初稿

把“段落结构 + 文献笔记 + 写作要求”一次性喂给 AI，要求：

- 仅使用你提供的文献素材
- 每段都要有引用占位 `\cite{}`
- 不编造结论与数值
- 行文保持“综述语气”，避免口语

---

## 二、可直接复用的 AI 提示词

```text
请根据我提供的文献笔记，撰写“1.2.3 基于雷达的生命体征感知研究现状”。

要求：
1) 共 5 段，结构为：背景概述 -> 基本原理 -> 传统研究 -> ECG重建 -> 不足与本文工作。
2) 每段都包含必要引用，使用 \cite{citekey} 形式。
3) 不要虚构论文结论，不要补充我未提供的数据。
4) 风格要求：中文学术论文表达，句子简洁，逻辑递进清晰。
5) 输出为 LaTeX 正文，不要输出 Markdown。
```

---

## 三、文献素材整理示例（推荐保留 Zotero 原始 BibLaTeX）

建议在素材文件中直接保留从 Zotero 复制的原始 BibLaTeX 条目，再在条目后追加你的中文笔记。  
这样可以减少手工整理工作量，也便于后续直接核对 `citekey`。

```md
@article{islam_RadarbasedRemote2022,
  title = {Radar-Based Remote Physiological Sensing: Progress, Challenges, and Opportunities},
  author = {Islam, Shekh Md Mahmudul},
  date = {2022}
}
该文用途：第一段背景综述，说明医学应用价值与挑战。

@article{paterniani_RadarbasedMonitoring2023,
  title = {Radar-Based Monitoring of Vital Signs: A Tutorial Overview},
  author = {Paterniani, Giacomo and others},
  date = {2023}
}
该文用途：第二段基本原理，说明相位解调与生命体征提取关系。

@article{yamamoto_ECGSignal2020,
  title = {ECG Signal Reconstruction via Doppler Sensor by Hybrid Deep Learning Model With CNN and LSTM},
  author = {Yamamoto, Kohei and others},
  date = {2020}
}
该文用途：第四段 ECG 重建早期代表方法。
```

可选补充模板（每条文献后追加）：

```md
- 问题：
- 方法：
- 结论：
- 在正文中的用途：
```

---

## 四、成稿后自检清单

- 是否严格按“5 段逻辑”组织？
- 每段是否有明确主题句？
- 是否存在“有结论无引用”或“有引用无论述”？
- 引用键是否都能在 `references.bib` 中找到？
- 最后一段是否自然过渡到“本文方法（如 Mamba-SSM）”？

---

## 五、实操建议

- 先求“结构对”，再求“句子美”。
- 文献笔记宁可短，不要复制大段摘要。
- AI 初稿完成后，务必手工改写为你的表达，并统一术语与符号。
